## usage

### 1.Installing

Add this to your package's `pubspec.yaml` file:

```yaml
dependencies:
  flutter_webviews:
    git:
      url: https://github.com/wafiqmuhaz/flutter_webviews.git
```

### 2.Import

```dart
import 'package:flutter_webviews/flutter_webviews.dart';
```
